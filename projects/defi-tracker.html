项目简介

DeFi收益追踪器是一个帮助你管理和分析DeFi投资收益的全能工具。 支持以太坊、BSC、OKChain等多个公链，自动追踪你的流动性挖矿、 Staking和借贷收益。

项目整合了Google NotebookLM的AI能力，可以根据你的投资历史生成个性化的策略建议， 让你的DeFi投资更加智能和高效。

主要功能

  • 多链支持：以太坊、BSC、OKChain等主流公链
  • 实时APY计算：自动计算并更新年化收益率
  • 收益趋势图表：直观的图表展示收益变化
  • AI策略分析：基于NotebookLM的智能投资建议
  • 价格预警：设定阈值，自动通知价格变动
  • 风险评分：评估DeFi协议的安全性
  • 多语言支持：中文、英文
  • 暗色/亮色主题切换

安装与使用

环境要求

  • Node.js >= 18.0.0
  • npm 或 yarn
  • MetaMask或其他Web3钱包

安装步骤

git clone https://github.com/zyerai/defi-tracker.git
cd defi-tracker
npm install

配置环境变量

cp .env.example .env.local
# 编辑 .env.local 添加你的API密钥

运行项目

npm run dev

访问 http://localhost:3000 查看应用
